# Project 01: Secure Azure VM Network

## Overview
This project demonstrates how to deploy a secure Azure Virtual Machine using networking and security best practices.

## Services Used
- Azure Virtual Machine
- Virtual Network (VNet)
- Subnets
- Network Security Group (NSG)
- Azure Storage Account

## Architecture
A VM is deployed inside a subnet within a virtual network. NSG rules are applied to control inbound and outbound traffic.

## Steps Performed
1. Created a Virtual Network and subnet
2. Deployed a Virtual Machine
3. Configured NSG rules to restrict access
4. Created a storage account
5. Verified secure connectivity
